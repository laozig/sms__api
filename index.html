API 概述

短信验证码API提供完整的短信验证码解决方案，让开发者能够轻松集成短信验证功能到自己的应用中。 我们的API服务稳定可靠，提供高并发支持，确保您的应用能够快速获取和验证短信验证码。

基础URL
http://127.0.0.1:5000/api
API特点
  • 提供多种功能完整的API接口
  • 支持多种运营商和号段类型
  • 高并发处理能力，平均响应时间低
  • 完善的用户管理和余额控制
  • 简单易用的RESTful API设计
性能指标

系统在中等并发负载下表现良好：

  • 根路由API：50并发用户下，RPS约115.82，平均响应时间12.48毫秒
  • 搜索项目API：50并发用户下，RPS约120.77，平均响应时间61.81毫秒
  • 获取手机号API：50并发用户下，RPS约40.73，平均响应时间585.73毫秒

认证方式

API使用基于Token的认证机制。在注册或登录成功后，系统会返回一个token，后续请求需要在参数中包含该token进行身份验证。

获取Token

通过以下两种方式获取token：

  1. 用户注册 - 注册成功后返回token
  2. 用户登录 - 登录成功后返回token
使用Token

在需要认证的API请求中，添加token参数：

GET /api/balance?token=YOUR_TOKEN
Token有效期

Token有效期为24小时，过期后需要重新登录获取新token。

用户注册

POST /api/register

注册新用户并获取访问token。用户注册后可以使用系统提供的各种API服务。

请求参数
参数名 类型 必填 描述
username string 是 用户名，唯一
password string 是 用户密码
email string 是 用户邮箱，唯一
security_question string 是 安全问题和答案，格式为"问题? 答案"
响应参数
参数名 类型 描述
success boolean 请求是否成功
message string 响应消息
user object 用户信息对象
user.token string 访问令牌
状态码
201 用户创建成功
400 请求参数错误或用户名/邮箱已存在
请求示例
GET /api/register?username=testuser&password=testpassword&email=test@example.com&security_question=测试问题?%20测试答案
响应示例
{ "success": true, "message": "用户注册成功", "user": { "id": 123, "username": "testuser", "email": "test@example.com", "balance": 0.0, "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VyX2lkIjoxMjMsImV4cCI6MTYzNDU2Nzg5MH0.abcdefghijklmnopqrstuvwxyz" } }

获取手机号

GET /api/get_phone

获取用于接收短信验证码的手机号。系统会根据请求参数返回一个可用的手机号，用户可以指定运营商类型和号段类型。

请求参数
参数名 类型 必填 描述
token string 是 用户访问令牌
project_id string 是 项目ID
carrier_type string 否 运营商类型(0:不限, 1:移动, 2:联通, 3:电信)
number_type string 否 号段类型(0:不限, 1:虚拟运营商, 2:非虚拟运营商)
响应参数
参数名 类型 描述
stat boolean 请求是否成功
message string 响应消息
code integer 状态码
data string 手机号
状态码
200 获取手机号成功
400 参数错误
401 未授权访问
404 找不到可用手机号
请求示例
GET /api/get_phone?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VyX2lkIjoxMjMsImV4cCI6MTYzNDU2Nzg5MH0.abcdefghijklmnopqrstuvwxyz&project_id=123456&carrier_type=0&number_type=0
响应示例
{ "stat": true, "message": "获取手机号成功", "code": 0, "data": "13812345678" }

获取短信验证码

GET /api/get_sms_code

获取指定手机号收到的短信验证码。系统会返回手机号收到的最新短信内容，其中包含验证码。

请求参数
参数名 类型 必填 描述
token string 是 用户访问令牌
project_id string 是 项目ID
phone string 是 接收验证码的手机号
响应参数
参数名 类型 描述
message string 响应消息
code integer 状态码
data string/null 短信内容或错误信息
状态码
200 获取验证码成功
400 参数错误
401 未授权访问
404 未收到验证码
请求示例
GET /api/get_sms_code?token=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VyX2lkIjoxMjMsImV4cCI6MTYzNDU2Nzg5MH0.abcdefghijklmnopqrstuvwxyz&project_id=123456&phone=13812345678
响应示例 - 成功
{ "message": "获取验证码成功", "code": 0, "data": "【淘宝】您的验证码是123456，5分钟内有效。如非本人操作，请忽略本短信。" }
响应示例 - 等待接收
{ "message": "暂未收到验证码，请稍后再试", "code": 1, "data": null }
